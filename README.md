# macropad_thai
Hackpad submission for Blueprint
the macropad it self is a 4x3 keypad with a knob and oled screen, i was inspired by adafruit industries rp2040 macropad.

I decided to not have a top case for style reasons, and definitely not because I got pissed off that my measurements were not working. Either way it would look cool to me.

# BOM
- 12x Cherry MX Switches
- 12x DSA Keycaps
- 12x 1N4148 DO-35 Diodes.
- 1x 0.91" 128x32 OLED Display
- 1x EC11 Rotary Encoder
- 1x XIAO RP2040
- 1x Case
